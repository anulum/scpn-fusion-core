# Public FRC Reference Data

This directory contains redistributable public Field-Reversed Configuration
(FRC) reference artefacts used for bounded validation and claim tracking.

## C-2U Positive-Net-Heating Table

`c2u_optometrist_positive_heating_shots.csv` is the supplementary shot table
for Baltz et al., "Achievement of Sustained Net Plasma Heating in a Fusion
Experiment with the Optometrist Algorithm", Scientific Reports 7, 6425 (2017),
DOI `10.1038/s41598-017-06645-7`.

The table reports C-2U shots with positive net heating power and includes
thermal energy, poloidal flux, total temperature, time of maximum net heating
power, maximum net heating power, and thermal energy at that time.

Claim boundary: this is public C-2U performance evidence. It is not Slough
2011 Fig. 5 trajectory data, not a time-resolved compression benchmark, and
not an external acceptance row for pulsed-compression parity.
